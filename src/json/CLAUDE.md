# JSON Directory

Static seed data for event categorization.

## Files

### event-categories.json
20 predefined event categories with descriptions. Used to seed the `event-categories` collection.

Categories: Entertainment, Educational, Sports, Relaxation, Cultural, Musical, Artistic, Networking, Outdoors, Food & Drink, Charity, Political, Religious, Family-Friendly, Adults-Only, Seasonal, Technology, Shopping, Travel, Health & Wellness

### event-types.json
~60 predefined event types, each with a name, description, and array of category references.

Examples: Rock Concert, Yoga Retreat, Hackathon, Wine Tasting, Charity Walk, Tech Conference, etc.
